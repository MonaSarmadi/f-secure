This is a simple application for Acceptance test
It uses Cucumber and Serenity and Java

The implementation of scenarios are in com.f_secure.features

It uses simple assertation

This application is not a show case for database interaction

steps to run the app locally:

clone the project or download it as a zip
open the intellij
import the project using pom.xml
go to runner class
click on the green play button

you have to put your chrome driver next to the pom.xml(make sure that the version of your chrome browser is compatible with your chrome driver)
